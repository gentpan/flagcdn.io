## 📖 概述

FlagCDN.io 是一个免费的 SVG 国旗图标 CDN 服务，基于 ISO 3166-1 alpha-2 国家代码。提供 4:3 和 1:1 两种比例的国旗 SVG 图标，支持 CSS 类名引用和直接 URL 访问。

---

## ✨ 特性

- 🏳️ **所有国家/地区** — 基于 ISO 3166-1 alpha-2 标准
- 📐 **双比例支持** — 4:3 和 1:1 两种宽高比
- 🚀 **CDN 加速** — 全球快速分发
- 📝 **CSS 类名引用** — 一行代码内联国旗
- 🔗 **直接 URL 访问** — 图片链接直接使用
- 🆓 **完全免费** — 无需注册，无使用限制

---

## 🚀 使用方式

### CSS 方式

```html
<link rel="stylesheet" href="https://flagcdn.io/flags.css">
<span class="fi fi-cn"></span>  <!-- 中国 -->
<span class="fi fi-us"></span>  <!-- 美国 -->
<span class="fi fi-gb"></span>  <!-- 英国 -->
```

### 直接 URL

```html
<!-- 4:3 比例 -->
<img src="https://flagcdn.io/cn.svg" alt="中国">
<img src="https://flagcdn.io/us.svg" alt="美国">

<!-- 1:1 比例 -->
<img src="https://flagcdn.io/w160/cn.png" alt="中国">
```
